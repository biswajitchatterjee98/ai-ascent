# AI Ascent: Topic & Subtopic Analysis

**Purpose:** Map every chapter before writing full content. Each topic targets **~2 pages of rich reading** (max 3 when comparison tables or checklists need space). Flow moves from foundations → models → generative craft → business outputs → risk/governance → tools → automation → program enrollment.

**Audience:** Working professionals and learners who want practical AI literacy, not only theory.

**Shared chapter pattern (used in the full handbook):**

1. Why it matters now 
2. Core concepts (step-by-step) 
3. Frameworks, comparisons, or deep dive 
4. Worked examples / scenarios 
5. Apply today (checklist or workflow) 
6. Key takeaways 
7. Media (topic image + 1-2 videos)
